HTML VS HTML5

HTML HTML 5
HTML stands for Hypertext Markup Language. HTML5 stands for Hypertext Markup Language 5.
HTML is the markup language used for creating web pages. HTML5 is the latest version of HTML and is also used for creating web pages.
HTML4 was the previous version of HTML. HTML5 introduced several new features and improvements over HTML4.
HTML4 had limited support for multimedia and required plugins for audio and video. HTML5 includes native support for audio and video elements, eliminating the need for plugins.
HTML4 had limited support for creating interactive web applications. HTML5 includes features like the canvas element and WebS
