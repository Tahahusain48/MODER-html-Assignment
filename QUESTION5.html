Sementic Elements

  • <header>: Represents a container for introductory content or a set of navigational links.
  • <nav>: Represents a section of a page containing navigation links.
  • <section>: Represents a thematic grouping of content.

These elements improve accessibility and help search engines better understand your content.

SVG(Scalable Vector Graphics)

HTML5 supports the <svg>element for rendering vector graphics. SVG graphics are resolution-independent and can be scaled without loss of quality. This is valuable for creating responsive and high-quality graphics.
